# shankara-a.github.io

Personal website of Shankara K. Anand, built with [Jekyll](https://jekyllrb.com/) and the [al-folio](https://github.com/alshedivat/al-folio) theme, hosted on [GitHub Pages](https://pages.github.com/).

## Local development

```bash
bundle install
bundle exec jekyll serve
```

The site will be available at `http://localhost:4000/`.

## Deployment

Pushes to `main` are automatically built and deployed to the `gh-pages` branch via GitHub Actions.
